# Angular E-Commerce & Product Management Application

A dynamic web application featuring complete user authentication, a shopping cart experience, and an administrative dashboard integrated with Cloud Firestore.

## 🚀 Key Features

*   **Custom User Authentication:** Secure signup, login, and route guarding via a dedicated `AuthService`.
*   **Firebase Firestore Integration:** Real-time CRUD (Create, Read, Update, Delete) operations handled through a centralized `AdminProductsService` for efficient catalog management.
*   **Dynamic E-Commerce Core:** Modular architecture featuring a product catalog, functional shopping cart system, and structured navigation via a dynamic layout header.

## 🛠️ Tech Stack

*   **Front-End:** Angular, JavaScript, RxJS, CSS
*   **Database & Auth:** Firebase Firestore & Firebase Authentication
