# Angular E-Commerce & Product Management Application

A dynamic web application featuring complete user authentication, a shopping cart experience

## 🚀 Key Features

*   **Custom User Authentication:** Secure signup, login, and route guarding via a dedicated `AuthService`.
*   **Dynamic E-Commerce Core:** Modular architecture featuring a product catalog, functional shopping cart system, and structured navigation via a dynamic layout header.

## 🛠️ Tech Stack

*   **Front-End:** Angular, JavaScript, RxJS, CSS
